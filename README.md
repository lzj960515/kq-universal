## 使用方式

使用该组件作为父工程或者单独使用其中的某个组件
```xml
    <parent>
        <groupId>com.github.lzj960515</groupId>
        <artifactId>kq-universal-starter-parent</artifactId>
        <version>1.2.0</version>
    </parent>
```

> 如果你的工程使用该组件作为父工程，那么你在使用该项目下的其他组件时，不再需要填写版号号
